# CherryFB

> ## 这是一个计算反馈电阻的程序。
>
> 选取`RES_0402_E96 电阻元件盒`中的电阻进行计算
>
> 省去在嘉立创扩展库选择巨贵电阻从而+20换料费的时间*（





用法：运行python



```python
##  +---- Vout   
##  |
##  | R1
##  |
##  +----FB      
##  |
##  | R2
##  +---- GND    
fb res:
```

会提示输入`Vfb`和`Vout`

```python
?Vfb=
?Vout=
```

一般可以有0.8v，1.0v，1.2v

vout看心情（



运行会选取`RES_0402_E96 电阻元件盒`中电阻进行计算，省去在嘉立创扩展库选择巨贵电阻从而+20换料费的时间*（
